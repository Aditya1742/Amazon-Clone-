# Amazon-Clone-
A static frontend clone of the Amazon homepage built using only HTML and CSS. This project is a visual replica focused on layout, responsiveness, and styling, without any backend or dynamic functionality.

Features-

1. Fully responsive Amazon-like homepage

2. Header with navigation bar and search box

3. Product cards and sections

4. Basic footer layout

5. CSS hover effects and transitions


Technologies Used

1. HTML5

2. CSS3 (Flexbox, Grid, Media Queries)
